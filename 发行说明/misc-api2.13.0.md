# misc-api 2.13.0 发行说明

**发布日期：** 2026-07-12  
**版本类型：** 大版本（前台 UI 全面调整）

## 概述

本版本对**默认主题（主题一）前台**进行全面重设计，参考独立 UI 稿的布局结构，但采用系统统一的浅色卡片风格（`--vs-*` 设计令牌），与用户中心、管理员后台保持一致。不包含参考稿中的极客绿、终端动效、API 文档页等系统未支持能力。

## 主要变更

### 默认主题前台

- **首页**：双栏 Hero、平台概览面板、统计数字动画、接口目录（搜索 + 分类 + 卡片网格）、平台特性区
- **全部接口**：完整接口列表，支持搜索、分类筛选、重置；展示已通过审核的公开 API / TAPI
- **子页面**：文章、关于、友链、贡献者、赞助统一 page-head + 卡片布局
- **页脚**：四栏链接 + 版权备案区；新增返回顶部按钮
- **交互**：移动端右侧抽屉保留；接口筛选与统计动画由 `theme.js` 驱动

### 核心能力

- `ApiManager::listPublic()` — 获取已通过审核的公开接口
- `ApiManager::countApproved()` — 统计公开接口数量
- `ApiManager::categoriesFromList()` — 从接口列表提取分类

### 主题包

- 默认主题版本：`1.3.0` → `1.4.0`
- 新增 `partials/api-grid.php`、`partials/api-toolbar.php` 复用组件
- **用户中心**（`user/`、`user.css`）未改动

## 升级说明

- **数据库：** 无结构变更（`db_changes: false`）
- 覆盖更新即可；若使用默认主题，刷新浏览器缓存（主题 CSS/JS 带版本号参数）
