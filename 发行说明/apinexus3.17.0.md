# ApiNexus 3.17.0 发行说明

**日期：** 2026-07-15  
**类型：** 中版本（用户 API 管理修复 + 列表/手机端体验）

## 下载

https://gitee.com/xunjinlu/apinexus/releases/download/v3.17.0/apinexus3.17.0.zip

## 变更

1. **用户中心 API 管理**  
   - 修复「提交接口」点击无反应、「编辑」报网络异常（图标选择器 API 应为 `setSelected`）  
   - AJAX 改用当前页完整 URL，兼容去 `.php` 伪静态

2. **状态展示**  
   - 已审核通过：列表只显示运行状态（正常 / 维护 / 禁用），不再叠「审核通过」  
   - 待审 / 不通过：仍显示审核态文案

3. **手机端**  
   - 用户 API 卡片操作按钮双列铺满  
   - 后台接口列表状态与操作区分行，避免挤成一团  
   - 默认主题用户中心标题与「提交接口」按钮对齐

## 库变更

无。
