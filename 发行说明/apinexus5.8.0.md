# ApiNexus 5.8.0 发行说明

**版本：** 5.8.0  
**日期：** 2026-07-22  
**类型：** 中版本（日志查询性能 + 冷热归档）  
**数据库：** 有结构变更（复合索引 + 配置项）

## 变更

- 后台「日志查询」默认只查近 N 天（默认 7，可调至 365），杜绝无条件全表 `COUNT(*)`
- `COUNT` 不再 `LEFT JOIN` 用户表；无筛选时总数走 Redis 短缓存，并与冷归档计数合并
- 翻页改为 keyset（`before_id`），避免深页 `OFFSET` 扫行
- 前端加载互斥 + `AbortController`，减少 F5/连点叠请求
- **冷热分层（重构，日志全部保留）**
  - 热数据：近 N 天（默认 30，系统设置可调）留在数据库
  - 冷数据：由计划任务归档到本机多级索引分片（按日目录 + ID 段文件）
  - 后台查询自动热冷合并，久远历史仍可查
- 系统设置 → API 日志：可生成计划任务密钥；无密钥调用归档入口返回无权
- 复合索引写入迁移：`(createtime,id)` / `(ok,createtime)` / `(apiid,createtime)`

## 升级注意

1. 后台执行 **数据库结构更新**（`install/migrations/5.8.0.sql`）
2. 到 **系统设置 → API 日志** 生成计划任务密钥，并在服务器配置每日凌晨调用（示例：`0 2 * * * curl -fsS "任务链接"`）
3. 大表首次加索引可能耗时，建议低峰操作
4. 下载：https://gitee.com/xunjinlu/apinexus/releases/download/v5.8.0/apinexus5.8.0.zip
