# ApiNexus 5.8.0 发行说明

**版本：** 5.8.0  
**日期：** 2026-07-22  
**类型：** 中版本（日志查询性能 + 冷热归档）  
**数据库：** 有结构变更（复合索引 + 配置项）

## 变更

- 后台「日志查询」默认只查近 N 天（默认 7，可调至 365），杜绝无条件全表 `COUNT(*)`
- `COUNT` 不再 `LEFT JOIN` 用户表；keyset 翻页；前端 Abort 互斥
- **冷热归档（可开关，日志全部保留）**
  - 热数据：近 N 天留在数据库
  - 冷数据：三层目录（总索引 / 日索引 / SQLite 分片，约 1000 条/片）
  - 高配服务器可关闭冷热分离，日志全留在线库
  - 后台查询自动热冷合并
- 系统设置 → API 日志：生成计划任务密钥；无密钥调用返回无权
- 复合索引写入迁移

## 升级注意

1. 后台执行 **数据库结构更新**（`install/migrations/5.8.0.sql`）
2. 若启用冷热归档：确保 PHP 启用 **PDO SQLite**；在系统设置生成密钥并配置凌晨计划任务
3. 大表首次加索引可能耗时，建议低峰操作
4. 下载：https://gitee.com/xunjinlu/apinexus/releases/download/v5.8.0/apinexus5.8.0.zip
