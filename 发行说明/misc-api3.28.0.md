# misc-api 3.28.0 发行说明

**日期：** 2026-07-17  
**类型：** 中版本（登录/发信安全 + 底栏扩展 + SEO + 库表变更）

## 下载

https://gitee.com/xunjinlu/misc-api/releases/download/v3.28.0/misc-api3.28.0.zip

## 变更

1. **忘记密码**  
   未在本站注册的邮箱禁止发送验证码（仍计入发信限流）。

2. **登录稳定性**  
   修复偶发「请求来源无效 / 请刷新重试」：同源校验放宽缺 Origin/Referer 场景，Host 规范化，会话寿命与 CSRF 回填。

3. **调用日志**  
   删除 `apilog.source`（来源类型）；跨域情况看「来源域名」等字段即可。

4. **主题二卡片**  
   列表不再预留描述空白，卡片高度降低。

5. **系统设置 · 站点扩展**  
   自定义底栏左/中/右 HTML、站点运行时间、1～2 个页脚二维码（名称 + 图片链接 + 启用）。

6. **双主题**  
   可开关：运行时间、页脚二维码；默认主题可开关合作伙伴；底栏精简并展示二维码。

7. **SEO**  
   公开页 meta / OG / Twitter；后台与登录 noindex；根目录 `robots.txt`。

8. **运行时间脚本归属（补丁）**  
   计时 JS 放在各主题 `assets/js/front-runtime.js`，由 footer 按需加载；已删除公共 `assets/js/front-runtime.js`。

9. **默认主题首页**  
   对 `$siteName` / `$siteDesc` / `$vsBase` / `$authUrl` 做注入兜底，消除缺变量告警。

## 升级注意

- **有数据库变更**：请在后台执行「数据库结构更新」（删除 `apilog.source`）。
- 强刷主题与后台静态资源。
- 运行时间、二维码、底栏代码请到 **系统设置 → 站点扩展** 配置；主题内再决定是否显示。
