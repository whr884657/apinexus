# misc-api 1.1.1 发行说明

**发布日期：** 2026-07-11  
**版本标签：** v1.1.1

## 本版摘要

优化用户后台 UI 与账号能力：自适应侧边栏布局、账号设置页、头像支持；官网隐藏管理后台入口。

## 变更内容

- 用户后台改为与管理员一致的自适应侧边栏 + 顶栏布局（PC/手机端）
- 新增 `/user/account.php` 账号设置（用户名、邮箱、头像、密码）
- 用户头像：QQ 邮箱自动提取 / 自定义图片链接 / 默认头像
- 用户表新增 `avatar_url` 字段
- 官网首页仅保留用户中心入口，管理后台需手动输入路径访问

## 数据库变更

- `vs_user` 表新增 `avatar_url` 列
- 迁移脚本：`install/migrations/1.1.1.sql`
