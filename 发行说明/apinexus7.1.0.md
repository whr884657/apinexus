# ApiNexus 7.1.0 发行说明

**版本：** 7.1.0  
**日期：** 2026-07-22  
**类型：** 中版本（内容运营体验 + 分类删除转移 + 主题色对齐）  
**数据库：** 有结构变更（`content.coverlayout`）

## 变更

- **Markdown 主题色**：引用竖线、卡片、时间轴、按钮跟随 `--accent-primary` / `--vs-primary` / `--st-accent`，不再写死绿色（E80）
- **公告管理**：编辑弹窗仅标题+正文；无摘要、无状态选择、无置顶/弹窗开关；保存即发布；置顶/弹窗在列表卡片操作；**关闭实时预览**；列表卡片对齐 API 列表（操作按钮右下）
- **文章管理**：封面布局左/右/背景；列表固定高度；电脑端实时预览、手机端隐藏预览
- **首页跑马灯**：公告用正文纯文本预览（无摘要业务）
- **接口分类**：删除时若仍有接口，电脑端大弹窗 / 手机端抽屉「转移至其他分类」后再删

## 升级注意

1. 后台执行 **数据库结构更新**（`install/migrations/7.1.0.sql`）
2. 下载：https://gitee.com/xunjinlu/apinexus/releases/download/v7.1.0/apinexus7.1.0.zip
