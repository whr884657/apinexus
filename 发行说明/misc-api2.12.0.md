# misc-api 2.12.0 发行说明

**发布日期：** 2026-07-12  
**类型：** 大版本（新模块 + 数据库变更 + 多主题优化）

---

## 变更摘要

### 后台 API 管理 · 接口审核

| 项目 | 说明 |
|------|------|
| 入口 | 后台侧边栏 **API 管理 → 接口审核** |
| 能力 | 待审核 / 已通过 / 已拒绝 / 全部筛选；通过、拒绝（填原因）、下线 |
| 数据 | 新增 `{prefix}api` 表，核心类 `core/ApiManager.php` |

### 主题设置修复与增强

- 修复保存主题后「当前使用」标志仍停留在旧主题（AJAX 响应字段读取错误）
- 主题卡片展示 `theme.json` 中的 **版本号**
- 主题配置表单支持 **select** 类型选项

### 青绿平台（slate）

- 主题设置新增 **手机端导航展开方式**：
  - `top_drawer`：顶部三横线 → 右侧抽屉（默认）
  - `fab_popup`：右下角圆形 FAB → 向上弹出（与用户中心一致）

### 默认主题（default）

- 首页 **完全移除「快速入口」** 板块及相关样式

## 升级说明

1. 覆盖代码后访问后台 **系统升级**，或确保 `install/migrations/2.12.0.sql` 已执行
2. 新表 `{prefix}api` 会自动创建，已有数据不受影响
3. 青绿 FAB 导航需在 **主题设置 → 主题设置** 中选择「右下角圆形按钮 · 向上弹出」并保存
