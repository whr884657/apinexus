# misc-api 2.1.0 发行说明

**发布日期：** 2026-07-11  
**类型：** 小版本（双主题 UI 优化 + 伪静态导航）

---

## 变更摘要

### 默认主题（default）

- 手机端侧边栏改为**从右侧展开**
- 移除抽屉顶部关闭按钮，点击遮罩空白处自动关闭
- 抽屉头部显示站点图标（跟随系统设置 Logo）与站点名称

### 云启风格主题（slate）

- 参考 API 平台浅色风格**全新重做**（绿色主色 `#24a66a`）
- **电脑端**：顶栏横向导航，不展开侧边栏
- **手机端**：右侧抽屉导航 + 遮罩点击关闭
- 首页、全部接口、文章、贡献者、友情链接、赞助、关于等配套页面统一风格
- 站点图标与名称跟随系统设置

### 伪静态导航

- 前台导航路径去除 `.php`（如 `/apis`、`/articles`）
- 登录/用户中心入口改为 `/user/login`、`/user/index`
- 需配置 nginx：`try_files $uri $uri/ $uri.php$is_args$args;`
