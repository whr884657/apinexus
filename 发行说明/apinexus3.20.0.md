# ApiNexus 3.20.0 发行说明

**日期：** 2026-07-16  
**类型：** 中版本（接口管理双端 UI 重构 + 统计注入简化）

## 下载

https://gitee.com/xunjinlu/apinexus/releases/download/v3.20.0/apinexus3.20.0.zip

## 变更

1. **统计注入更简单**  
   - 推荐 3 行向上查找 `core/bootstrap.php`  
   - 备选：已知深度 `dirname`；`api` 下一层可用 `require_once dirname(__DIR__) . '/hit.php';`  
   - 详见 `api/统计代码使用说明.md`

2. **接口列表 / 审核 / 用户 API 管理 UI**  
   - **电脑端**：表格行布局（非大卡片）  
   - **手机端**：参考卡片（图标+编号+名称、右上角标签、GET/链接、右下角请求次数、底部操作条）  
   - 标签：分类、免费、KEY可选/必填、代理（仅管理端）、状态（正常/维护/禁用互斥）

3. **规范**  
   - 《开发规范与功能优化》写入「代码宜简不宜繁」强制条款

## 升级注意

- 无数据库结构变更。  
- 本地接口请按新版说明更新头部统计代码（旧复杂写法仍可用，建议改成 3 行）。

## 库变更

否。
