# misc-api 3.27.0 发行说明

**日期：** 2026-07-17  
**类型：** 中版本（全页配色纠偏 + 详情页 UI + 统计/设置体验）

## 下载

https://gitee.com/xunjinlu/misc-api/releases/download/v3.27.0/misc-api3.27.0.zip

## 变更

1. **主题二全页配色**  
   切换浅色后，页面背景、顶栏、底栏、抽屉、搜索条等与默认浅绿同结构的渐变一并变色。

2. **首页统计**  
   用户数 / 今日调用等所有 `.st-stat-num` 都会执行数字动画（不再只动三项）。

3. **调色盘图标**  
   与管理员后台调色盘同款 SVG。

4. **主题设置**  
   统计开关改为左右双列网格。

5. **接口详情页**  
   默认主题与主题二均重做详情 UI：面包屑、信息卡、参数/返回/文档区块、复制地址；手机端自适应。

## 升级注意

- 无数据库结构变更。  
- 强刷主题二与默认主题静态资源。  
- 详情地址仍为 `/detail.php/{id}`（PATH_INFO）。

## 库变更

否。
