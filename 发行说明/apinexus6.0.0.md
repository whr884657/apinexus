# ApiNexus 6.0.0 发行说明

**版本：** 6.0.0  
**日期：** 2026-07-22  
**类型：** 大版本（贡献者生态 + 公开个人主页）  
**数据库：** 有结构变更（用户简介 / 博客 / 壁纸 + 全站默认壁纸配置）

## 变更

- **贡献者页**：展示已发布公开接口的开发者真实名片（头像、接口数、累计调用、加入时间）；卡片可点击进入个人主页
- **公开个人主页** `/profile/{id}`（默认主题与主题二）：
  - 毛玻璃资料卡、发布接口列表、搜索排序
  - 背景壁纸：用户自定义优先，否则使用系统设置中的全站默认图（可填随机图接口）
  - 接口卡片延迟：经 `core/ping.php` 检测主机连通耗时
- **账号设置**：可填简介、博客、主页背景
- **系统设置 → 站点扩展**：可配置全站默认个人主页背景

## 升级注意

1. 后台执行 **数据库结构更新**（`install/migrations/6.0.0.sql`）
2. 伪静态已有通用 `/{页}/{数字ID}` 规则时，无需再改 Nginx（`/profile/1` 自动可用）
3. 下载：https://gitee.com/xunjinlu/apinexus/releases/download/v6.0.0/apinexus6.0.0.zip
