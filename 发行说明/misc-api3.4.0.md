# misc-api 3.4.0 发行说明

**发布日期：** 2026-07-14  
**版本类型：** 大版本增量（`3.3.0` → `3.4.0`，多模块 UI / 体验批量优化）  
**数据库变更：** 否

---

## 概述

本版本统一两主题认证页输入提示方式，修复 Toast 单行垂直居中，增强 Redis 管理页实时计时（刷新周期文案保持静态），并调整用户管理与接口分类手机端卡片布局。

---

## 认证页

| 变更 | 说明 |
|------|------|
| 主题 1 | 登录 / 注册 / 忘记密码 / 绑定：移除上方 label，改用 placeholder + `aria-label` |
| 主题 2 | 注册用户名 placeholder：`请输入用户名（3～50 个字符）`（补全「用户名」语义） |

---

## Toast

- 长条使用 flex 垂直居中 + 内层 `.vs-toast__text`，单行文案不再偏上

---

## Redis

- 运行时长：天 / 小时 / 分 / 秒，页面内每秒累加（无需刷新）
- 业务缓存项：「刷新周期」与键名文案保持静态；仅「剩余 N 秒」实时倒计时，归零后显示「未缓存」

---

## 后台卡片

- **用户管理（手机）**：角色标签在右上角、QQ/Gitee 图标左侧；卡片右下（操作按钮上方）显示最后登录时间
- **接口分类（手机）**：接口数量在右上角「启用/禁用」标签左侧
