# misc-api 3.26.0 发行说明

**日期：** 2026-07-17  
**类型：** 中版本（主题二体验 + 后台分页 + 可配统计/配色）

## 下载

https://gitee.com/xunjinlu/misc-api/releases/download/v3.26.0/misc-api3.26.0.zip

## 变更

1. **后台分页**  
   「每页」横向排列；页码选择框与「上一页 / 下一页」同高（修复 vs-pick 占满宽度导致竖排）。

2. **主题二接口卡片**  
   首页卡片可进入详情；同时展示 GET/POST；方法标签左对齐；卡片固定高度，描述占位、调用地址贴底。

3. **主题二首页统计（后台可配）**  
   可分别开关：分类数、接口数、注册用户、今日调用、累计调用。

4. **主题二配色**  
   后台可设默认浅色（默认浅绿）；前台顶栏圆形调色盘可选调色板第 5～12 项淡色（本地记忆）。

## 升级注意

- 无数据库结构变更。  
- 使用主题二时：后台 → 主题设置 → 配置统计项与默认色。  
- 强刷静态资源缓存。

## 库变更

否。
