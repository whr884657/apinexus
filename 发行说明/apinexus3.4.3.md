# ApiNexus 3.4.3 发行说明

**发布日期：** 2026-07-14  
**版本类型：** 小版本（`3.4.2` → `3.4.3`，紧急修复 Redis 图表 UI）  
**数据库变更：** 否

---

## 概述

回退 3.4.2 引出线饼图方案（实测在手机端线条溢出、整体错乱），恢复为 3.4.1 稳定的环形图交互效果，并加强卡片防溢出。

---

## 修复

| 项 | 说明 |
|----|------|
| 回退 | 移除扇区引出折线标注实现 |
| 恢复 | SVG 环形图：悬停/点击高亮扇区，明细在图下方提示区展示 |
| 防溢出 | 图表卡片 `overflow: hidden`；提示文案折行 |
