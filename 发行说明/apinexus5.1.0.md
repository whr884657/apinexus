# ApiNexus 5.1.0 发行说明

**版本：** 5.1.0  
**日期：** 2026-07-20  
**类型：** 中版本（SEO 全站 + UI + 性能调优）

## 亮点

1. **SEO 全站增强**
   - `vs_render_seo_meta()` 扩展 OG / Twitter / Schema itemprop / 移动端 meta
   - 分享图、canonical、og:url 统一转绝对 URL（提升 QQ / 微信抓取稳定性）
   - 各前台公开页、后台、用户中心均输出独立 title + description

2. **默认主题 · 详情页推荐接口**
   - 去除板块内嵌套卡片背景，内容区更宽
   - endpoint 链接框样式与首页一致
   - 推荐逻辑改为同分类优先 + 按调用量排序（不再随机）

3. **首页接口目录**
   - 固定按 calls 降序、id 降序展示前 8 条

4. **在线测试稳定性**
   - 浏览器直连增加 30s 超时、最多 3 次重试、`cache: no-store`

5. **API 日志开关**
   - 系统设置 → API 日志：开启写 apilog；关闭仅累加 `api.calls`

6. **Redis 缓存精简**
   - 友链、合作伙伴、分类、前台接口列表等小数据直读 MySQL
   - 保留 apilog 查询 / 今日统计等大数据短时缓存

## 升级

1. 覆盖代码文件
2. 后台「系统升级」检查版本 → 5.1.0
3. **无需**数据库结构更新
4. 建议在「系统设置 → API 日志」确认日志策略

## 数据库变更

否
