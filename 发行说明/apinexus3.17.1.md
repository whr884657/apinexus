# ApiNexus 3.17.1 发行说明

**日期：** 2026-07-15  
**类型：** 小版本（认证：忘记密码 / 修改密码稳定性）

## 下载

https://gitee.com/xunjinlu/apinexus/releases/download/v3.17.1/apinexus3.17.1.zip

## 变更

1. **忘记密码**  
   - 邮箱查找与会话比对统一小写，避免大小写差异导致「邮箱或验证码错误」  
   - 重置成功后不再因 MySQL `rowCount=0`（新旧密码哈希相同）误报失败  
   - 主题页与管理员页：响应 JSON 解析加固，减少误报「网络异常」

2. **账号设置改密（用户中心双主题共用 `user/account.php`）**  
   - 用户名长度改按 Unicode「字」计，修复中文用户名保存失败  
   - 头像输入改为 `text` + `novalidate`，并放宽合法链接校验（http/https 或站内 `/` 路径）  
   - 修改密码前端先校验再提交，避免含糊失败

3. **架构说明**  
   - 两套主题仅渲染 UI；验证与改密均经 `user/*.php` → `UserAuth` / `Auth` 核心层访问数据库，主题不直连字段

## 库变更

无。
