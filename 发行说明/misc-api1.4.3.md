# misc-api 1.4.3 发行说明

**发布日期：** 2026-07-11  
**类型：** 小版本（账号设置 UI 修复）  
**数据库变更：** 无

## 本版摘要

修复用户/管理员账号设置页 Notice 提示条显示不一致：一条占满整行、另一条与输入框错位。

---

## 变更详情

- 「修改密码」区块提示移入 `vs-form-row--notice`，与输入框同列
- 字段级提示（如邮箱找回密码说明）宽度与输入框一致（`max-width: 420px`）
- 本地文档《界面提示规范.md》新增 §4.1 账号表单对齐约定
