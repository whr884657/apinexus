# misc-api 3.7.0 发行说明

**发布日期：** 2026-07-14  
**版本类型：** 大版本（图标库扩展 + 后台接口列表体验）  
**数据库变更：** 否

---

## 概述

将新增 SVG 图标全面纳入系统图标库，并在接口分类 / 接口列表弹窗中可搜索选用；强化接口列表双端交互与空状态引导。若站点仍显示「功能开发中」，请升级到本版（或至少 3.6.0）并确认已打开 `/admin/api/list.php`。

---

## 变更

| 项 | 说明 |
|----|------|
| 图标库 | `assets/img/category-icons/` 扩充至约 188 个合规 SVG |
| 选择器 | `ApiCategoryManager` 自动扫描；`icon-picker.js` 搜索 / 懒加载 / 入场动画 |
| 接口列表 | 空状态引导、升级提示、行与 Tab 过渡动画；手机端适配 |
| 性能 | 页面仅下发图标相对路径，前端拼接站点 Base |

---

## 升级说明

- 无数据库迁移；覆盖或在线升级即可。
- 建议强刷后台缓存（`?v=3.7.0`）。
- 若从未执行过 3.6.0 库表迁移，仍需先完成结构更新后再使用接口列表。
