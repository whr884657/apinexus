# misc-api 3.32.0

## 变更说明

- 请求参数「类型」改为嵌套选择器：电脑端在编辑弹窗内再开次级弹窗，手机端在编辑抽屉内再开底部抽屉（不完全盖住父层）
- 类型以自适应网格展示（英文 + 中文），并支持自定义类型输入
- 修复原先 vs-pick 下拉被表格容器裁切、几乎看不见选项的问题（E28）

## 升级说明

- 无数据库变更，覆盖更新即可
- 建议清理浏览器缓存后验证管理员/用户中心「提交/编辑接口」参数类型选择
