# misc-api 3.22.0 发行说明

**日期：** 2026-07-17  
**类型：** 中版本（接口列表桌面行式重构 + 分页/统计外置）

## 下载

https://gitee.com/xunjinlu/misc-api/releases/download/v3.22.0/misc-api3.22.0.zip

## 变更

1. **电脑端**  
   接口列表/审核/用户 API 管理改为紧凑**行式**布局，不再用大卡片堆叠。

2. **手机端**  
   收紧面板与卡片左右留白；操作按钮改为扁矮（约 22px 高），单行横排。

3. **分页与统计**  
   - 分页移出白色背景板，居中显示页码序号（1 / 2 / 3…）+ 上一页/下一页  
   - 「每页」条数选择移到分页区  
   - 「当前接口总数…」从标题旁移到页底居中（分页下方）

4. **文案**  
   去掉提示中的「列表按接口编号从大到小排列」。

5. **元信息顺序**  
   靠右：提交者在最右，请求次数在其左侧。

6. **开发规范**  
   明确禁止用临时 Python/补丁脚本改业务文件；临时功能性文件任务结束必须删除，禁止提交。

## 升级注意

- 无数据库结构变更。  
- 请强刷后台/用户中心以加载新 CSS/JS。

## 库变更

否。
