# misc-api 2.8.0 发行说明

**发布日期：** 2026-07-12  
**类型：** 中版本（青绿导航与前台视觉重构）

---

## 变更摘要

### 用户中心

| 项目 | 说明 |
|------|------|
| 侧栏 | **完全移除**左侧侧栏 |
| 导航 | 右下角**圆形三横线**按钮，点击后**向上弹出**菜单 |
| 菜单项 | 控制台、API 管理、令牌管理、积分变动、接口列表、账号设置、退出登录 |
| 选中态 | 仅青绿背景高亮，**无左侧绿色竖线** |

### 前台首页（青绿平台）

- 整体背景改为**浅青绿渐变**（`st-root`）
- 顶栏、统计胶囊、搜索框、分类标签、卡片、页脚统一青绿调
- 悬停效果改为阴影/渐变，而非仅绿色边框
