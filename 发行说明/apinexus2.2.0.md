# ApiNexus 2.2.0 发行说明

**发布日期：** 2026-07-12  
**类型：** 中版本（用户中心主题联动）

---

## 变更摘要

### 用户中心主题联动

- 登录、注册、忘记密码、OAuth 绑定页视图迁入 `core/theme/{id}/user/auth/`
- 用户中心 shell 迁入 `core/theme/{id}/user/layout.php`
- 切换前台主题后，用户中心界面同步切换（管理员后台不受影响）

### 主题规范文档

- 新增根目录 **[主题规范.md](../主题规范.md)**：目录结构、自动扫描、资源隔离、新增主题步骤

### 后台主题设置

- 「当前使用」徽章**仅在已保存主题**上显示，选择未保存时不跟随变化
- 移除冗余测试阶段提示语

### 青绿平台首页

- 收录接口 / 今日调用 / 累计调用：**三项独立卡片**，从左往右横排

### 预览图说明修正

- 预览图为开发阶段随主题包预部署，非站长自行放置
