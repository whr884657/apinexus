# ApiNexus 5.9.0 发行说明

**版本：** 5.9.0  
**日期：** 2026-07-22  
**类型：** 中版本（订单/积分查询性能 + 冷库分片可配）  
**数据库：** 有结构变更（orders 复合索引 + 配置项）

## 变更

- 调用日志冷库：**每个分片条数可配置**（默认 5000，系统设置 → API 日志）
- 管理员「订单管理」「积分变动」、用户中心「积分变动」：
  - 默认只查近 30 天；keyset 翻页（最新一页起往后翻）
  - 取消深页 OFFSET；COUNT 不再带多余 JOIN
  - 充值订单列表不再 JOIN 接口/密钥表
  - 前端 Abort，减少连点叠请求
- 积分流水**不做冷热归档**，全部留在数据库
- 新增复合索引，加速时间窗与按用户/状态查询

## 升级注意

1. 后台执行 **数据库结构更新**（`install/migrations/5.9.0.sql`）
2. 大表首次加索引建议低峰操作
3. 下载：https://gitee.com/xunjinlu/apinexus/releases/download/v5.9.0/apinexus5.9.0.zip
