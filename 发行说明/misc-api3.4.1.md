# misc-api 3.4.1 发行说明

**发布日期：** 2026-07-14  
**版本类型：** 小版本（`3.4.0` → `3.4.1`，Redis 监控交互与去冗）  
**数据库变更：** 否

---

## 概述

优化 Redis 缓存监控页：去掉与图表重复的图例与统计卡片，改为可交互 SVG 环形图；缓存项状态图默认展示缓存占用，并改善多端布局。

---

## Redis 监控

| 变更 | 说明 |
|------|------|
| 去冗 | 移除三图下方重复图例；移除命中/未命中/命中率/占用四宫格（与图重复） |
| 交互 | 悬停/点击扇区高亮，下方提示区显示「命中：N 次（x%）」等明细；再次点击取消固定 |
| 缓存占用 | 「缓存项状态」图中心默认显示业务缓存占用 |
| 多端 | ≥1100px 三列、中屏两列、手机单列；遵循 `prefers-reduced-motion` |
