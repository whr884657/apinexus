# ApiNexus 5.7.0 发行说明

**版本：** 5.7.0  
**日期：** 2026-07-21  
**类型：** 中版本（主题设置新能力 + 规范完善）  
**数据库：** 无结构变更

## 变更

- 后台「主题设置」新增 **累计调用次数显示**：
  - **完整数字**：实时有多少显示多少（不转换单位）
  - **单位转换**：达到千/万后显示为 K+、W+（默认，保持原观感）
- 默认主题与主题二首页均支持；仅作用于「累计调用」一项
- 完善《主题规范》《开发规范与功能优化》中分页 / 布局 / 配色专项债与版本号判断约定

## 升级注意

- 无数据库迁移
- 升级后请到 **系统管理 → 主题设置** 按需调整「累计调用次数显示」
- 下载：https://gitee.com/xunjinlu/apinexus/releases/download/v5.7.0/apinexus5.7.0.zip
